# 📖 Overview

The **Cyber Security Foundation** project is a comprehensive hands-on learning experience designed to build core cybersecurity knowledge and practical skills. This project focuses on understanding cybersecurity principles, Linux fundamentals, networking concepts, system security, risk management, and security operations through real-world laboratory exercises.

The lab environment was completed using **ALNAFI Cloud**, providing practical exposure to industry-relevant cybersecurity tools, techniques, and workflows.

---

# 🎯 Project Objectives

Upon completion of this project, learners gain the ability to:

✅ Understand core cybersecurity concepts

✅ Identify common cyber threats and attack vectors

✅ Apply security best practices

✅ Navigate and administer Linux systems

✅ Analyze network communications

✅ Understand authentication and access control

✅ Perform basic security monitoring

✅ Implement security hardening techniques

✅ Understand risk management principles

✅ Develop a cybersecurity mindset
